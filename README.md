# Tarun Kumar — Portfolio

Cinematic portfolio with a 3D character scene, scroll-driven sections, and data-driven content.

## Run

```bash
npm install
npm run dev
```

## Main content

Update copy, links, work items, and related fields in `src/data/site.ts` and `src/data/techStack.ts`.

## 3D assets

Uses encrypted character data in `public/models/` together with the HDR environment and Draco decoder files.

## Images

Visuals live under `public/images/`.
